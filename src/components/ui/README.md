# UI Components
This folder contains "dumb" or "presentational" components. These are the building blocks of your app.

- They should be reusable.
- They generally don't have their own "State" (except for things like being open/closed).
- They receive data via **props**.

## Examples:
- `Button.jsx`
- `DataTable.jsx`
- `Form.jsx`
- `Skeleton.jsx`
